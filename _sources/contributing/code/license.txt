Licencia *Symfony2*
===================

*Symfony2* se libera bajo la licencia *MIT*.

De acuerdo con `Wikipedia`_:

    "Es  una  licencia  permisiva,  lo cual  significa  que  permite  la
    reutilización dentro de  software propietario a condición  de que la
    licencia se distribuya  con el software.  Esta también es compatible
    con  la licencia  *GPL*, lo  cual  significa que  la licencia  *GPL*
    permite la combinación y redistribución, con el software que utiliza
    la licencia *MIT*".
