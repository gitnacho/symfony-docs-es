ContentBundle¶

Este paquete proporciona un documento para contenido estático y el controlador para dibujarlo.

Para más información —por ahora— ve la documentación del ContentBundle

Configurando¶

La clave de configuración para este paquete es symfony_cmf_content

  • YAML
    # app/config/config.yml
symfony_cmf_content:
    admin_class:          ~
    document_class:       ~
    default_template:     ~
    content_basepath:     /cms/content
    static_basepath:      /cms/content/static
    use_sonata_admin:     auto
    multilang:
        admin_class:          ~
        document_class:       ~
        use_sonata_admin:     auto
        locales:              []
