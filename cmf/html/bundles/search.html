SearchBundle¶

El SearchBundle proporciona integración con LiipSearchBundle para manejar la búsqueda en un sitio grande.

Dependencias¶

  • LiipSearchBundle

Configurando¶

La clave de configuración para este paquete es symfony_cmf_search

  • YAML
    # app/config/config.yml
symfony_cmf_search:
    document_manager_name:  default
    translation_strategy: child # además se puede configurar como una cadena vacía o atributo
    translation_strategy: attribute
    search_path: /cms/content
    search_fields:
        title: title
        summary: body
