Configurando WebProfiler¶

Configuración predeterminada completa¶

  • YAML
    web_profiler:

    # DEPRECATED, it is not useful anymore and can be removed safely from your configuration
    verbose:              true

    # mostrar la barra de depuración web en la parte inferior de las
    # páginas con un resumen de información del perfil
    toolbar:              false
    position:             bottom
    intercept_redirects:  false
